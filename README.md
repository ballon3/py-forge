
# Py-Forge!

## Description

Just a Python web application using Autodesk Forge Web APIs

## Prerequisites

    * Python 3.x

    * MongoDB

## Project Setup

    * cd <directory containing this project>

    * python -m venv .

    * ./bin/pip install -e .

    * ./bin/pserve development.ini --reload
